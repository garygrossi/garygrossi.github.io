3D Geographic Data Visualization

About

GeoCorre is a 3D data correlations visualization tool set. It allows multi-dimensional data analysis by representing
correlation between data sets over a geographic region. This tools has many uses including understanding socio-economic
data relationship, viewing the political interest of an area, and determining relationships between medical problems,
location, and other factors. It also includes Leap Motion functionality to allow for natural motion based data navigation.
This project was created at Philly{Codefest} 2014 by team Atomic Dragons.
